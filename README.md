## How to use?

- Clone GitHub repository:
```
git clone https://github.com/Kerso-official/MmcliSmsClient && cd MmcliSmsClient
```
- Start up the python script:
```
python3 main.py
```
- Annddd you're all done :3! (use 1-6 commands to interact with the app)
